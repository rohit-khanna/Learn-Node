The Log files will be created at this location.

- All the unhandledExceptions will be captured in "unhandledExceptions.log" log file.
- All the other handled Error Logs will captured in "errorOnly.log" log file.
- All the logs - including error/warninig/info/debug/verbose  will be captured in "allLogs.log" log file.
     (This file can be used as data source of investigation)
